# Arduino i2c arm

![Arm for small robot dog](https://github.com/SovGVD/arduino-i2c-arm/blob/main/assets/img/small_dog_arm.jpg?raw=true)

## 4DOF with gripper
 - Board: Arduino Mini
 - Servos: 5 x WK-P0025
 - ToF: VL53L1X
 - DC-DC: TBC

## Features
 - [ ] Control arm via i2c
 - [ ] ToF sensor (return image)
 - [ ] UART CLI for setup and control
 - [ ] Save seting into EEPROM
